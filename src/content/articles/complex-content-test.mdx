## Code Blocks

Syntax highlighting is essential for technical blogs. Here are some examples:

```javascript
// JavaScript example
function calculateFactorial(n) {
  if (n === 0 || n === 1) {
    return 1;
  }
  return n * calculateFactorial(n - 1);
}

console.log(calculateFactorial(5)); // 120
```

```typescript
// TypeScript example
interface User {
  id: number;
  name: string;
  email: string;
}

class UserService {
  private users: User[] = [];
  
  constructor() {}
  
  public addUser(user: User): void {
    this.users.push(user);
  }
  
  public getUserById(id: number): User | undefined {
    return this.users.find(user => user.id === id);
  }
}
```
